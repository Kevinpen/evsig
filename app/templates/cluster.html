Cluster

The Yeast_elife_2019 IDR evolutionary signature data were clustered using Cluster 3.0 program (de Hoon et al., 2004). The resulting files can be viewed by the TreeView program. For instructions on how to visualize the clusters, please consult the "Java TreeView User's Manual" in http://jtreeview.sourceforge.net/docs/JTVUserManual/single.html
